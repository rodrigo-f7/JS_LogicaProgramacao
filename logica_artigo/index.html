O que é um Algoritmo?

Antes de falar sobre o algoritmo é interessante falar sobre a lógica. A lógica é a capacidade de raciocinar de maneira ordenada levando uma conclusão. Existem alguns tipos de lógica, como a lógica dedutiva, a lógica indutiva e a lógica abdutiva.

A lógica dedutiva é aquela que com base em uma preposição consegue extrair novas conclusões de acordo com exemplos. Por exemplo: Todo homem é anão. Sócrates é homem, logo Sócrates é anão

Já a lógica indutiva busca identificar uma regra geral ou um padrão com base em uma série de exemplos. Por exemplo: está chovendo agora de manhã, logo pode continuar chovendo à tarde

Por último a lógica abdutiva é a lógica qe se relaciona com a observação e a determinação de hipóteses, capaz de criar conhecimentos novos e realizar descobertas.
Exemplo: Você vê uma pessoa com raiva na rua, assim você pode pensar nas inúmeras possibilidades que o deixariam com raiva (hipóteses), ao perceber que ele está com uma maleta e vestido de terno você pode pensar que ele está com raiva do trabalho, e isso é ainda mais confirmado quando ele está entrando em um prédio com agitação.

Assim, podemos chegar ao conceito de algoritmo:

Algoritmo é um conjunto finito de passos que levam a uma solução contra um problema.

Saida de dados

A saida de dados na lógica de computação é a capacidade de exibir dados, processados ou não, em um dispositivo periférico do computador

Variáveis

Variáveis são espaços endereçados no computador que armazenam valores, como números, booleanos, strings e outros valores.

Entrada de dados

A entrada de dados na lógica de programação são instruções que permitem o programa detectar dados externos, comumente inseridos por teclados ou sensores.

Operadores

Operadores são símbolos que possibilitam realizar operações entre operandos, como números ou caracteres. Os mais comuns são os operadores aritméticos (+, -, /, *), os operadores relacionais (<, >, ≠ =, >=, <=, etc.) e os operadores lógicos (&& E, || OU, ! NEGAÇÃO)

Estruturas condicionais

Estruturas condicionais são marcadores lógicos que modificam o fluxo de execução do programa, possibilitando saídas e resultados diferentes de acordo com as condições atendidas ou não atendidas.

Existem as estruturas condicionais simples, compostas, aninhadas e os switch-case (Escolha-caso).

Primeiros comandos em JavaScript

Para começar em Javascript, basta você ter um computador com um navegador instalado e um editor de texto de sua confiança como o VScode ou o Sublime Text.

Todo primeiro código de uma linguagem é o famoso "Hello, World!", um programa que exibe um dado na tela, normalmente por um terminal.

console.log("Hello, World!") document.write("Hello, World!")

Estes são comandos simples de saída de dados em Javascript. console.log() realiza a impressão de texto na console web, já o document.write realiza a escrita diretamente no documento html. Vale ressaltar que o código javascript precisa ser escrito na tag <script> ou chamado a partir dessa tag com o atributo src.

Variáveis em JavaScript

JavaScript é conhecida como uma linguagem fracamente e dinamicamente tipada. Isto significa que não há verificação de tipo antes da execução do código e possibilita que os tipos de dados de uma variável sejam alterados.

Assim, algumas formas de declarar variáveis estão presentes a seguir:

x = 5; x = "Salvador" var nome = "João" let possuiCartao = true; const botao = document.querySelector(".botao")

Nesse caso, há 4 formas de se declarar uma variável em JavaScript. Na primeira e segunda linha, sem definir o escopo, também percebe-se a alteração do dado da variável para um do tipo string, algo totalmente possível em Javascript.

Em seguida, há a declaração por meio de var, let e const.

Será explicado mais adiante sobre a definição específica dessas palavras-chave, com uma exceção para const, que basicamente define que é uma constante, e não uma variável, não podendo ter seu valor alterado.

Operadores em JavaScript

Os operadores em Javascript são semelhantes ao da linguagem Java e outras linguagens. A seguir os principais operadores da linguagem

Operadores Aritméticos JS

Operador Função
"+" Soma ou concatena duas expressões. Se colocado diante de um número determina qual o seu sinal (positivo)
"-" Subtrai duas expressões

Bônus:

  • numero.toFixed(2): realiza o arredondamento do número em questão para uma qunatidade de casas decimais
  • let soma = array.reduce(function (soma, i) { return soma + i; }): método .reduce executa uma série de operações em cada um dos elementos do array em questão, nesse caso está fazendo uma soma
